Wabbajawn

The Wabbajawn is a weapon mod for the 2023 sci-fi game Starfield, developed by Bethesda Game Studios.

Game Development Mod Release Mod Author
Game Overview

The mod introduces a new weapon that, when fired at an NPC, turns them into randomized hostile creatures for 30 seconds.

Info

The weapon (for now) uses a vanilla gun asset in the base game as the basis for the Wabbajawn weapon. The effect is managed by a script written in Papyrus - the proprietary scripting language used in Bethesda/Creation Engine Games.

Background

The weapon was inspired by another iconic weapon, the Wabbajack, from another Bethesda game series (The Elder Scrolls), which had a similar effect.

There are eventual plans to release this mod on Bethesda's mod management site for Xbox release
